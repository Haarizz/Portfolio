# HAARIZZ — Harikrishnan T Nair · Portfolio
## Antigravity-style

A clean, light, Google-Antigravity-inspired design: white background, an animated
field of drifting colored dashes (Google brand colors), soft sans-serif typography,
pill buttons, and a giant centered hero. Single-file static site, vanilla JS.

## Files
- `index.html` — the entire site
- `Harikrishnan-T-Nair-Resume.pdf` — resume
- `vercel.json` — clean URLs, caching, security headers

## Run locally
Open `index.html`, or `python -m http.server 8000`.

## Deploy (Vercel)
Framework Preset **Other**, no build command, output dir `.`.

Interactions: animated dash field (mouse-repelling), hero entrance, scroll reveals,
animated counters. Respects `prefers-reduced-motion`.

> One of six portfolio designs. See the hub at `/`.
