# HAARIZZ — Harikrishnan T Nair · Portfolio
## Studio (flagship)

A dark, cinematic editorial flagship that fuses four aesthetics into one:
- **Agency / award** — big serif display type, gradient headline, animated marquee
- **Apple** — sticky-scroll "Process" section where panels pin and activate as you scroll
- **Linear / Spotify** — app-style work cards with per-project accent colors + filters
- **Editorial** — typographic about section with a pull-quote and live counters

Single-file static site, vanilla JS. Aurora background, grain overlay, scroll
progress, active-nav, magnetic buttons, reveal-on-scroll. Respects
`prefers-reduced-motion`.

## Files
- `index.html` — the entire site
- `Harikrishnan-T-Nair-Resume.pdf` — resume
